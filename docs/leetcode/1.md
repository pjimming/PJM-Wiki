#### 题目描述：

找出数组中重复的数字。

在一个长度为 n 的数组 nums 里的所有数字都在 0～n-1 的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。

#### 示例 1：
```
输入：
[2, 3, 1, 0, 2, 5, 3]
输出：2 或 3 
```

#### 限制：

`2 <= n <= 100000`

---

#### 思路：
新开一个集合用于存储出现过的数，遍历该数组，每次记录当前的数，如果在该集合内，即为答案。
