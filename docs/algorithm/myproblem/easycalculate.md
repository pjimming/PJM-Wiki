# 一个简单的计算题

[原题链接](https://yzcode.top/problem/T1005)

#### 题目描述
请计算 $\sum_{i=l}^{r}\left\lceil\sqrt[k]i\right\rceil$

其中 $\left\lceil x\right\rceil$ 表示向上取整，$\sqrt[k]x$ 即为对 $x$ 开 $k$ 次方根。

由于答案可能过大，请答案对 $10^9+7$ 取模。

#### 输入格式
输入占一行，三个整数，分别为 $l,r,k$。

#### 输出格式
一个整数，即计算得到的结果。

#### 数据范围
- $1\le l,r\le 10^{12}$
- $1\le k\le 10$

#### 输入样例1
```
1 20 2
```

#### 输出样例1
```
70
```

#### 输入样例2
```
100 300 3
```

#### 输出样例2
```
1264
```
